# K-Spam: 한국 특화 스팸 필터 앱

개발자가 답답해서 만든 스팸 필터앱!

스팸으로부터 자유로워지세요!

## 기능
1. 기본 탑재 Spam필터
2. 해외발신 필터
3. 화이트/블랙리스트 필터 커스텀 기능
4. 카지노 충전 문자 필터
5. 수신동의 광고 차단 필터
